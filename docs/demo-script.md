# Demo Script

## Success path
1. Start system
2. Submit teaching requirement
3. View pending requirements
4. Search available teachers
5. Assign teacher to requirement
6. Record training
7. Exit system
8. Restart system
9. Verify saved data still exists

## Failure path
1. Start system
2. Search or assign a teacher whose qualification does not match
3. Verify UI prints a clear error message
